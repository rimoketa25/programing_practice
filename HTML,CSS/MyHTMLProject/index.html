<pre>タグの使い方

改行や空白もそのまま反映されてしまうので、preタグ内は字下げをしないようにする。

divタグとspanタグの使い方

divタグ：行のブロックからなる領域のスタイルを変えるために使う
spanタグ：文中の一部の要素のみに対してスタイルを変える
